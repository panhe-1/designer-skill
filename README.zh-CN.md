# Designer Skill

[English README](README.md)

Designer Skill 现在是一个适合公开分享、同时注意隐私安全的 skill 源仓库。

这个仓库保留的是可复用的 skill 内容，不再公开那些带有个人信息或由 skill 后续生成出来的项目产物。

## 当前保留的内容

- `SKILL.md`
  正式的 skill 入口文件，说明这个 skill 适合在什么场景下使用，以及应遵循的核心原则。
- `Claude-Design-Sys-Prompt_中文翻译(1).txt`
  作为详细参考材料保留的中文翻译提示词正文。
- `README.md`
  英文版仓库说明。
- `README.zh-CN.md`
  中文版仓库说明。

## 明确不上传的内容

- 用这个 skill 生成出来的 PPT、PNG、预览导出图
- 论文或其他带个人信息的源文档
- 写死了本地路径、个人身份信息或具体项目内容的个性化脚本

## 为什么现在是这个结构

这是有意为之。仓库的目标是公开保留 skill 本体和说明，而不是把你后续实际项目里的私人上下文一起暴露出去。

## 说明

- `SKILL.md` 现在已经补上，仓库不再只是一个文本快照。
- `Claude-Design-Sys-Prompt_中文翻译(1).txt` 继续作为完整参考材料保留。
- 本地后来生成的导出目录、PPT 成品和 `.docx` 文档都已经加入忽略规则，避免误上传。
