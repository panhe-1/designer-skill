# Designer Skill

[English README](README.md)

Designer Skill 是一个面向设计场景的 skill，用来生成更完整、更有设计感的视觉交付物。

## 功能介绍

- 生成高保真 HTML 原型和界面 mockup
- 支持偏幻灯片风格的视觉表达和演示内容设计
- 用于探索排版、配色、层级、动效和交互表现
- 能结合现有产品、品牌、截图或设计系统来统一风格
- 支持多方向设计迭代和细节调整

## 使用场景

当任务目标不是单纯写代码，而是需要更强的界面设计、原型设计、视觉表达或展示设计时，这个 skill 会更适合。
