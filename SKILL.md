# Designer Skill

This skill is for design-heavy tasks that need stronger visual quality, presentation structure, and iterative refinement.

## Use This Skill When

- the user wants high-fidelity HTML prototypes, interactive mockups, slide-style deliverables, or visual concept work
- the work should align with an existing codebase, product UI, screenshot set, brand language, or design system
- the user wants multiple visual directions, tweaks, or iterative design exploration rather than a single static answer

## Core Function

- build polished design artifacts with clear hierarchy, typography, spacing, motion, and interaction
- match the existing visual language before introducing new stylistic directions
- explore multiple options when the task benefits from comparison and iteration
